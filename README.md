# SariSari

Fullstack Web-based daily sales tracker for small sari sari stores.

## Website link
- [SariSari Tracker](https://sarisari-tracker.netlify.app/)

## Technology

- React JS
- Node JS / Express JS
- MongoDB
- Tailwindcss
- Zustand [state manager]
